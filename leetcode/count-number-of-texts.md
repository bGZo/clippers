## Conclusion

为什么 36 个 2 最后能得到 2082876103 种结果，而不是 $3^{36}$?

```
222222222222222222222222222222222222
```

**每一个按键只会映射三个字母，用户选择字母的时候，会连续摁下数次某一个按键，但是每个按键的上线是 3 次。**
